🚀 Excited to share my latest project during my internship at Innomatics Research Labs!

I built a Retrieval-Augmented Generation (RAG) Application that allows users to:
🔗 Input any website link
❓ Ask questions about the content
⚡ Get precise, context-aware answers instantly

💡 Key Features:
Web scraping using LangChain
Semantic search with vector databases (ChromaDB)
Google Gemini LLM integration
Streamlit-based interactive UI
Secure API handling using environment variables
🔥 Deployed the application on AWS EC2 for real-time access

This project helped me strengthen my understanding of:
📌 Generative AI
📌 NLP pipelines
📌 Cloud deployment (AWS EC2)
📌 Building production-ready Generative AI applications
